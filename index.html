FAQ Sales

Soalan biasa sebelum mula projek website.

Berapa lama website boleh siap?

Biasanya 3-7 hari bekerja selepas bahan asas lengkap seperti logo, servis, lokasi, gambar dan nombor WhatsApp diterima.

Domain dan hosting termasuk?

Boleh disediakan atau dibantu setup mengikut pakej. Jika client sudah ada domain, website boleh disambungkan ke domain tersebut.

Saya belum ada logo, boleh buat website?

Boleh. Kami boleh gunakan wordmark asas dahulu atau bantu cadangkan logo ringkas untuk permulaan.

Boleh letak ToyyibPay?

Boleh jika website perlu terima bayaran online. Untuk website servis biasa, CTA WhatsApp sudah mencukupi pada fasa awal.

Boleh update website selepas siap?

Boleh. Update kecil atau penambahan section boleh dibuat mengikut keperluan dan scope support.

Website sesuai untuk iklan?

Ya. Landing page boleh digunakan sebagai link utama untuk Facebook Ads, TikTok Ads, Google Ads, Google Business Profile dan media sosial.
