# RH Services Documentation

Dokumentasi repo disusun dalam folder berikut:

- `architecture/` — struktur sistem, database, routing dan keputusan teknikal utama.
- `qa/` — checklist dan keputusan QA.
- `updates/` — nota perubahan teknikal mengikut modul.
- `releases/` — nota release/version.

Root repo dikhaskan untuk fail runtime utama sahaja.
