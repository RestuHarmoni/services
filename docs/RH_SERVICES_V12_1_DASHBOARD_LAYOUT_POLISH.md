# RH Services v12.1 Dashboard Layout Polish

Tujuan patch ini ialah menyusun semula dashboard admin supaya lebih jelas sebagai Sales Workspace.

## Perubahan utama

- Dashboard diposisikan sebagai RH Services Sales Dashboard.
- Workflow bawah dashboard ditukar daripada Office Workflow kepada Sales Workflow.
- Aliran rasmi: Lead → Prospect → Quotation → Negotiation → Won Project.
- Customer ditukar kepada Prospect kerana customer sebenar hanya selepas deal menang.
- KPI dashboard disusun kepada New Leads, Prospects, Quotation Pipeline, Negotiation dan Won Projects.
- Quick Actions ditambah: Lead Inbox, Prospect File, Create Quotation.

## Tidak diubah

- Aira lead capture.
- Blog admin.
- Template pakej.
- Supabase schema.

Patch ini layout/UI only dan tidak memerlukan SQL baru.
