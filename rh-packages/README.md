# RH Package Templates

This folder aligns the uploaded package templates with official RH package names.

| Official Package | Source Template | Purpose |
|---|---|---|
| RH Starter | old Pakej 1 | Company Profile Website |
| RH Growth | old Pakej 2 | Service & Product Website |
| RH Ecosystem | old Pakej 3 | Company Profile + Service Website / Subdomain ecosystem |

Aira must recommend only these official names:

- RH Starter
- RH Growth
- RH Ecosystem

Legacy labels such as Package 1, Package 2, Package 3, RH Basic, RH Standard, RH Premium are not used for new leads.
